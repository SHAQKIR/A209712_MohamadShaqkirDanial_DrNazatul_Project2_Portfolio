💭 Reflection & Learning Journey

Starting from Lab 1 where I first built basic UI with Compose, to Project 2 where I integrated GPS sensors, live APIs, and cloud databases — the growth has been significant.

The biggest challenge was understanding how Room, DataStore, Firebase, and the API all work together in one ViewModel without causing data conflicts. Using a single AppState data class and collecting multiple Flows using combine() was the solution that made everything click.

EcoWatt contributes to SDG 7 by giving users real-time awareness of their energy footprint and connecting them with a community of like-minded energy savers. Technology can genuinely drive sustainable behaviour change.
